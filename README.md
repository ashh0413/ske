# Skeletron Music Website

A static, responsive Skeletron artist/academy website inspired by the structure and mood of nikisadeki.com.

## Pages
- `index.html` — home, about, academy, music, newsletter
- `biography.html` — artist story and gallery
- `music.html` — listening platforms, releases, press cards
- `videos.html` — video/social showcase
- `booking.html` — booking, academy, promo and general inquiries

## Run Locally
```powershell
cd "D:\Development\PROJECTS\SKE2"
python -m http.server 8123
```

Open `http://localhost:8123/index.html` in a browser.

## Notes
- Source photos are in `Photos/`.
- Optimized web assets are in `assets/images/`.
- Update email addresses and social links before publishing if different from the placeholders used here.
